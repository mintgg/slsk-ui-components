FormInput

FormInput

表单组件
支持受控值、非受控默认值、前缀节点、后缀节点、最大输入长度、清空按钮、回车事件和校验状态的基础输入框组件。

组件预览

元
当前受控值：可编辑的受控值
最近回车值：-

基础用法

import { FormInput } from "@/src/components/form-input";

export function Demo() {
  return (
    <FormInput
      placeholder="请输入内容"
      onChange={({ value, event }) => console.log(value, event)}
      onChangeValue={(value) => console.log(value)}
      onPressEnter={(event) => console.log(event.currentTarget.value)}
    />
  );
}

Props

字段名：value
描述：受控值
类型：string
默认值：-
字段名：defaultValue
描述：非受控默认值
类型：string
默认值：-
字段名：disabled
描述：是否禁用
类型：boolean
默认值：false
字段名：hideClear
描述：是否隐藏清空按钮
类型：boolean
默认值：false
字段名：prefix
描述：输入框前缀节点
类型：React.ReactNode
默认值：-
字段名：suffix
描述：输入框后缀节点
类型：React.ReactNode
默认值：-
字段名：placeholder
描述：输入框占位文本
类型：string
默认值：-
字段名：maxLength
描述：最大输入长度
类型：number
默认值：-
字段名：status
描述：校验状态，用于错误态或警告态展示
类型："error" | "warning"
默认值：-
字段名：onChange
描述：值变化时触发
类型：(payload: InputChangePayload) => void
默认值：-
字段名：onChangeValue
描述：值变化时触发，仅返回当前值
类型：(value: string) => void
默认值：-
字段名：onPressEnter
描述：回车时触发
类型：React.KeyboardEventHandler<HTMLInputElement>
默认值：-
